# 🔐 OPEN Password Extension

A powerful Chrome extension to generate secure, customizable passwords instantly—right from your browser.

---

## 🚀 Features

* Generate strong, random passwords
* Customize length and character types
* Copy to clipboard with one click
* Password strength indicator
* Save password history
* Dark / Light mode
* Lightweight and fast

---

## 🖥️ How It Works

Click the extension icon → generate a secure password instantly without leaving your current page.

---

## 📦 Installation (Developer Mode)

1. Download or clone this repository
2. Open Chrome and go to:
   `chrome://extensions/`
3. Enable **Developer Mode**
4. Click **Load unpacked**
5. Select the project folder

---
